# Выводы

Реализован мультипликативный конгруэнтный генератор (MCG) с параметрами:
- **M = 2³¹ − 1**
- **BETA = x0 = 48271**

## Результаты (выборка: 100 000 значений)

| Показатель | Теоретическое | MCG | std::mt19937_64 |
|------------|--------------|-----|----------------|
| **Среднее** | 0.5000000000 | 0.5007827438 | 0.4990357055 |
| **Дисперсия** | 0.0833333333 | 0.0833357743 | 0.0832485964 |
| **Отклонение среднего** | — | 7.8×10⁻⁴ | 9.6×10⁻⁴ |
| **Отклонение дисперсии** | — | 2.4×10⁻⁶ | 8.5×10⁻⁵ |

## Заключение

1. Реализованный MCG демонстрирует статистические характеристики, близкие к теоретическим значениям для равномерного распределения U[0,1).

2. Отклонения выборочных оценок от теоретических значений составляют менее 0.1% для среднего и менее 0.003% для дисперсии — что соответствует ожидаемой точности при размере выборки 10⁵.

3. Качество генерации реализованного MCG сопоставимо со стандартным генератором `std::mt19937_64`, а по точности дисперсии даже превосходит его.

4. Простая формула `xᵢ = (a·xᵢ₋₁) mod M` при корректно подобранных параметрах обеспечивает достаточное качество псевдослучайной последовательности для задач, не требующих криптографической стойкости.

> **Итог:** Мультипликативный конгруэнтный метод с параметрами Парка-Миллера является работоспособным и эффективным базовым генератором псевдослучайных чисел.

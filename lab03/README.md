### Клеточные автоматы. Лесные пожары (GUI)

**Задание:**  
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата.

**Требования:**
- реализовать **не менее трёх дополнительных правил** поведения системы.

**Правила клеточного автомата**  
Модель представляет собой синхронный клеточный автомат размером 150×100. Каждая клетка имеет тип (CellType) и возраст (Age). Возможные состояния: Empty (пустая земля), Tree (дерево), Burning (горит), Water (вода).  

**1. Базовые правила**  
Выгорание: клетка полностью выгорает за один шаг.  
Распространение огня: дерево загорается, если среди соседей (8 направлений) есть огонь.  
Самовозгорание: дерево загорается с базовой вероятностью f = 0.0005 (например, из-за молнии).  
Восстановление: дерево вырастает на земле с вероятностью p = 0.05.  

**2. Дополнительные правила**  
Для усложнения модели реализовано три дополнительных правила:  
Водоемы: При инициализации в центре карты создается прямоугольная зона Water. Вода не горит, не меняет состояние и блокирует распространение огня на соседние клетки.  
Векторный ветер: Направление ветра (windDir) влияет на вероятность заражения от соседей:  
Если горящий сосед находится по направлению ветра — возгорание происходит гарантированно (100%);  
Для остальных направлений — вероятность снижена до 30%.  
Возрастная уязвимость: Каждое дерево накапливает возраст (Age). Если возраст превышает порог oldAgeThreshold = 20, вероятность случайного самовозгорания для такой клетки возрастает в 10 раз (старые деревья легче загораются). Визуально старые деревья отображаются тёмно-зелёным цветом.

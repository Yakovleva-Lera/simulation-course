### Моделирование случайных событий (GUI)  

**Часть 1:**  
Приложение «Скажи “да” или “нет”».

**Часть 2:**  
Приложение «Шар предсказаний» (Magic 8-Ball).

## Описание  
Приложение демонстрирует моделирование случайных событий с неравномерными вероятностями на платформе Windows Forms. В основе реализации лежит генератор случайных чисел, использующий мультипликативный конгруэнтный метод без применения встроенной генерации случайных чисел.

Выбор случайного исхода выполняется методом сравнения сгенерированного числа с заданными вероятностями. Генератор инициализируется фиксированным значением (seed = 52) для обеспечения воспроизводимости результатов.
